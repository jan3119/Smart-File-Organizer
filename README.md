# 🗂 Smart File Organizer (Python GUI Automation Tool)

A Python-based desktop automation tool that organizes files automatically by type, detects duplicates using hashing, and provides a GUI interface for easy interaction.

---

## 🚀 Features

- 📂 Automatically organizes files by type (Images, Documents, Videos, Music)
- 🔁 Duplicate file detection using MD5 hashing
- 🖥 GUI built with Tkinter
- 📝 Activity logging with timestamps
- ⚡ Multithreaded execution (non-freezing UI)
- 📊 Real-time progress tracking
- ⏰ Supports automation via Windows Task Scheduler

---

## 🛠 Tech Stack

- Python
- Tkinter (GUI)
- OS & Shutil modules
- Hashlib (MD5)
- Multithreading
